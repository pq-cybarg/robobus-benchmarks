Read the layers

The stack spans nine orders of magnitude and each layer has its own regime — conflating them is how benchmarks lie. The runtime hot path is ns/GB·s; handshakes are a one-time per-peer cost amortised to zero over a session.

one bus · every transport

Bridges everything — and secures the links that can't secure themselves

robobus sits at the centre as a post-quantum-secured hub; a thin bridge maps each protocol's topics or streams onto bus channels. And any bridge takes --security cnsa20 --keyfile — so CNSA 2.0 crypto wraps MQTT, Kafka, serial or CAN exactly as it wraps DDS, even where the wire protocol has no security of its own.

reach it from any language

Six native implementations, fourteen generated

The wire format is small — an mmap seqlock ring, a codec frame, an AES-256-GCM frame, fixed-layout schema structs. Six languages implement it natively and are bidirectionally conformance-tested against Python (Python seals → the other opens, and back); the schema layout is code-generated for fourteen more, each compile/run-verified against Python's packed little-endian bytes. Every one of these toolchains is installed and run green together — no language is ever silently skipped.

Embed it anywhere — librobobus. A dependency-light C-ABI shared library (robobus.dll · librobobus.dylib · librobobus.so, built by CMake on every OS) exposes AES-256-GCM RBX1 seal/open + the codec, so software in any language links robobus over its FFI. Proven byte-identical to the Python reference on Linux, macOS, Windows and five emulated ISAs — including big-endian s390x.

crypto

CNSA 2.0 post-quantum

FIPS 203 ML-KEM and FIPS 204 ML-DSA, hybridised with classical ECDH/ECDSA through HKDF-SHA384 — the NSA CNSA 2.0 transition profile, not a toy.

transport

Sub-microsecond bus

A lock-free shared-memory SPSC ring in C: single-digit-nanosecond ops, p50 one-way IPC under 100 ns, formally proven (CBMC) and RT-hardened.

dds

PQC-hardened DDS-Security

Patched CycloneDDS & Fast DDS authentication: hybrid ECDH+ML-KEM key agreement and ML-DSA-87 identity signatures, measured live on Linux, macOS and Windows.

Every language toolchain installed and run green together — 6 native seal/open bindings plus 14 schema-codegen targets, no silent skips; the C-ABI shared library proven byte-identical on six ISAs incl. big-endian s390x; CBMC + SymbiYosys proofs of the ring.

honest measurement

How the numbers are made

Nanosecond timing uses amortised batching (a single clock_gettime on macOS is only ~1 µs granular); a 20k-message warmup is discarded so cold-start jitter doesn't pollute the steady-state tail; DDS handshakes are reported as the isolated FSM window (begin → OK), not the ~100 ms process wall-clock (startup + SPDP discovery + teardown).

Coverage, honestly. The public site's CI runners are virtualized (shared vCPUs, no real-time scheduling), so their columns are for cross-platform coverage and relative comparison — not bare-metal nanosecond claims. Each run self-labels its measurement fidelity.
